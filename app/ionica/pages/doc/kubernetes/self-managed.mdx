# Self Managed Kubernetes using NixOS

Learn how to set up a self-managed Kubernetes cluster using NixOS on your own
Virtual Private Server (VPS).

## 1. Setup

### 1.1 Purchase a VPS

To begin, purchase a Virtual Private Server (VPS) from your preferred provider.
Ensure the VPS has a minimum of 6GB of RAM to effectively run NixOS and Kubernetes.

### 1.2 Connect to the server

Use VNC or SSH to connect to the server. Follow the instructions provided by
your VPS provider.
